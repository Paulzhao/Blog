MSSQL SSL Security Error

2020-06-13

#MSSQL SSL Security error

今天ruby脚本的执行的时候遇到了下面的错误。

分析了一下，这个可能是跟公司禁用TLS1.0有关系，尽管我在注册表里打开了TLS1.0 还是不能解决这个问题，看错误信息是ODBC去连接的时候SSL的检查不能通过，可能是DB那边的设置不允许TLS1.0的连接，但是老的ODBC的配置还是用的TLS1.0. 于是就去把机器上的ODBC升级了一下，并且更新了连接字符串，完美解决。

ODBC17 链接

https://www.microsoft.com/en-us/download/details.aspx?id=56567

同时发现了下面这个老版本SQLCMD的问题，安装了最新版的SQLCMD就好了.

下面是SQLCMD的链接

https://docs.microsoft.com/en-us/sql/tools/sqlcmd-utility?view=sql-server-ver15（这个需要先安装 .Net Framework4.5 和 ODBC 17)
